What is TypeScript?

  • Extends JavaScript:
    • Standard Types: string, number
    • Tuples
    • Enums
    • `Any` Type
    • Nullable Types
  • Statically typed.
  • Strongly typed - with type inference.
  • Gives access to new features:
    • Classes
    • Interfaces
    • Namespaces / Modules
    • Union types
  • Transpiled to Vanilla Javascript
      Allows developers to use new features while targeting previous versions of JavaScript.

Why use TypeScript?

  • QOL Improvements
    • Code completion / assist
    • Compile time errors, faster feedback cycle
    • Syntactical sugar - C# inspired
  • Community Support
    • Created by Microsoft
    • Adopted by Google for Angular
    • Definition libraries - .d.ts files for most javascript libraries
